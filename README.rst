The underlying process management library for `xonsh`_.

Example
-------

.. code:: python

    with ProcessGroup() as pg:
      pipe = Pipe()
      spam = pg.add(Process(['spam'], stdout=pipe.side_in))
      eggs = pg.add(Process(['eggs'], stdin=pipe.side_out))
    pg.start()
    pg.join()

.. _xonsh: http://xon.sh/
